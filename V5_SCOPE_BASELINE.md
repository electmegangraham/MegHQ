# V5 Scope Baseline

Defines what is included in V5.

## In Scope
- Core object model
- Validation + authority enforcement
- Task / initiative execution
- Approval + artifact system
- Megan’s Desk + Staff Mode
- Core intelligence (routing, alerts)

## Out of Scope (V5.1+)
- Extended automation layers
- Advanced analytics dashboards
- Non-critical integrations
