# Social Media System

## Purpose
Social media must operate like a real campaign department, not just a posting utility.

---

## Core functions

### Content creation
- plan content
- generate drafts
- align with doctrine
- tie content to goals

### Monitoring
- track comments
- track DMs
- track mentions
- track reactions and performance
- detect issues, supporters, and opportunities

### Engagement tracking
The system should identify:
- supportive engagement
- questions
- attacks
- issue signals
- community opportunities

### Narrative alignment
Social should be tied to:
- doctrine
- campaign narrative
- strategy
- field realities
- current priorities

### Performance feedback loop
Social should feed back:
- what messaging works
- what topics resonate
- what underperforms
- what needs adjustment

This feedback should influence:
- messaging
- doctrine
- strategy
- field and outreach coordination

---

## Long-term need
This department will eventually need:
- content objects
- scheduling/publishing logic
- sentiment logic
- escalation rules
- response routing
- integration with CRM and Campaign OS
