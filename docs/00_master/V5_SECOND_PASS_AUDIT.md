# 3. Taste / Instinct Layer (Critical)

This is still the biggest “could be lost” category if not made explicit.

Must be preserved as enforceable truth:

## 3.1 We Loved
- calm authority
- structured clarity
- trust-first design
- above-the-fold clarity
- strong hero
- professional systems language
- real-world language
- simple grounded wording

## 3.2 We Rejected
- playful / bubbly design
- loud campaign feel
- fluff language
- overbuilt visuals
- dashboard theater
- fake CRM behavior
- feature sprawl
- generic political phrasing

## 3.3 Final Taste Rule
If it feels like a campaign, it is probably wrong.
If it feels like a well-run office, it is right.

Gap:
These are present in source corpus but not yet fully formalized as system-enforced rules.
